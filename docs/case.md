### expand()
- 트리 순회 (트리에 있는 필드를 순회) -> 토큰
	
1. find metacharacter for expansion in field : $, *
	except : `''$PATH''[o], '$PATH'[X], '''$PATH'''[X], ''''$PATH''''[O]`
    except : `"'$PATH'"`
2. if $, do env expand, `export a="*"; ls $a`의 결과물이 `ls *`의 결과물과 동일
3. if *, do filename expand
